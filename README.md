# Armyworm
  
          
## 简介
###*Armyworm*是一款用于研究和测试的IE密码输入控件黏虫防护能力的测试程序

##开发环境
    Microsoft Visual Studio 2013 / MFC
    
##使用说明
    1.编译程序
    2.运行Armyworm，点击开启黏虫
    3.打开IE，转到包含密码输入控件的网页
